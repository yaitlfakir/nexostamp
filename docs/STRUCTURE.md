## Quick Reference

### Keyboard Shortcuts
- `Ctrl+N` - New design
- `Ctrl+O` - Open design
- `Ctrl+S` - Save design
- `Ctrl+Z` - Undo
- `Ctrl+Y` - Redo
- `Ctrl+D` - Duplicate
- `Ctrl+P` - Print preview
- `Del` - Delete selected

### Common Tasks
1. **Add Text**: Click "T" button
2. **Add Shape**: Click Rectangle/Circle/Line button
3. **Select Element**: Click on it (yellow border appears)
4. **Move Element**: Click and drag
5. **Resize Element**: Change Width/Height in Properties
6. **Rotate Element**: Use Rotation slider
7. **Change Text**: Type in Text field
8. **Change Font**: Select from Font Family dropdown
9. **Make Bold**: Check Bold checkbox
10. **Save Work**: Ctrl+S or File â†’ Save
